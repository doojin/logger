Golang Logger
=============

[![Build Status](https://travis-ci.org/doojin/logger.svg)](https://travis-ci.org/doojin/logger)

Usage
=====

The simplest way to use logger:
    import "./logger"
    
    func main() {
        logger.Info("Hello, world!")
    }
Will output:
>15:52:57 [INFO] Hello, world!
